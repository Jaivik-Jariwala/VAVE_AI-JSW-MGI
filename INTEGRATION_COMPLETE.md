## How It Works

### API Integration
```python
# Automatically uses Gemini API if available
# Falls back to mock data if API key not set
from vave_presentation_engine import generate_deep_dive_ppt

# Generate presentation (uses real LLM)
generate_deep_dive_ppt(ideas_list, output_path)
```

### Environment Setup
The engine automatically detects:
- `GOOGLE_API_KEY` environment variable
- `GEMINI_API_KEY` environment variable (alternative)

If neither is set, it uses mock data (useful for testing).

### Data Format Compatibility
The engine accepts ideas in multiple formats:
- **Old format**: `{'Cost Reduction Idea Proposal': '...', 'Saving Value(INR)': '...'}`
- **New format**: `{'title': '...', 'raw_description': '...', 'saving_amount': ...}`
